# qrcode_generator

跨平台的二维码生成工具

## 用法

```
PS C:\mygo\qrcode_generator\bin> .\qrcode_generator_windows_amd64.exe
Usage of C:\mygo\qrcode_generator\bin\qrcode_generator_windows_amd64.exe:
  -border int
        二维码周围的空白边框大小（模块数） (default 4)
  -logo string
        要嵌入到二维码中间的图片路径
  -logo-size int
        logo图片的尺寸（像素），建议不超过二维码尺寸的1/3 (default 50)
  -output string
        输出的二维码图片文件名 (default "qrcode.png")
  -size int
        二维码图片的尺寸（像素） (default 256)
  -text string
        要转换为二维码的文本

```
